# HOW TO BUILD A STANDALONE APK
# Two methods — choose whichever works for you

## METHOD 1: EAS Build (Easiest — cloud build, no setup needed)
# This builds the APK in Expo's cloud servers. FREE for 30 builds/month.

Step 1 — In Termux:
  pkg install nodejs -y
  cd ~/derivpro
  npm install
  npm install -g eas-cli

Step 2 — Create free Expo account at expo.dev then login:
  eas login

Step 3 — Build APK (takes ~10-15 min, done in cloud):
  eas build --platform android --profile preview

Step 4 — When done, EAS gives you a download link.
  Download the .apk and install it on your phone.
  Enable "Install from unknown sources" in Android settings first.

Done — fully independent .apk, no Expo Go, no Termux needed to run it.
